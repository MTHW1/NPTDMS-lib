Находящаяся в данном репозитории библиотека nptdms является аналогом официальной одноименной библиотеки с фиксом проблем, связанных с расширенными числами (extended float).

Установка:
1) Удалите оригинальную библиотеку nptdms, если она установлена.

2) Установите исправленную версию вручную в свою среду разработки.
